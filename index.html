🎯 产品介绍

CS-AI Agent是一个基于人工智能的智能客服系统，专门为Shopify商家设计。通过Gmail Add-on的形式，为您的客服团队提供强大的AI辅助功能。

🧠 智能意图分析

自动识别客户邮件的意图类型（WISMO、退款、其他），提取关键信息如订单号、产品信息等。

✍️ 智能回复生成

基于邮件内容和Shopify数据，自动生成个性化的回复草稿，支持多种回复风格。

🛍️ Shopify数据集成

实时查询订单信息、物流跟踪、退款状态和客户历史，为回复提供准确数据支持。

🔒 企业级安全

零持久化架构，邮件内容仅在内存中处理，通过Google Cloud确保数据安全。

🚀 如何开始使用

  1. 安装Gmail Add-on
    在Google Workspace Marketplace中搜索并安装CS-AI Agent
  2. 授权Shopify连接
    在Gmail中打开Add-on，完成Shopify商店的OAuth授权
  3. 开始处理邮件
    打开客户邮件，点击右侧边栏的CS-AI Agent图标
  4. 审查AI建议
    查看AI分析的意图和生成的回复草稿
  5. 发送回复
    确认无误后发送回复，或根据需要进行编辑
